Flashing Instruction for iqraOS V3.1
📦 Requirements
- Windows PC / Laptop
- USB Cable
- Extracted ROM folder on Desktop
- Device in Fastboot Mode

🚀 Flashing Procedure
- Extract the downloaded ROM zip file to your Desktop.
- Open the ROM folder and locate FlashWindows.bat.
- Double-click FlashWindows.bat to launch the flasher.
- The tool will ask for a flashing preference:
- y → Clean Flash (Formats Data – Recommended for best stability)
- n → Dirty Flash (Keeps existing user data)
- Wait patiently while the flashing process completes.
- Do not disconnect the device during flashing.
Once finished, the device will reboot automatically.

🔄 First Boot Information
Initial boot can take several minutes.
Boot delay is normal; do not force restart or panic.
💡 Notes
A clean flash is strongly recommended when switching base versions.
If you encounter boot issues, reflash using the Clean Flash option.
Enjoy the experience — iqraOS | Based on HyperOS
